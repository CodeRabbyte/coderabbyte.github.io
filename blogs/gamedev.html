Game Development

❮ ❯

Fast pace mecha shooter with lockon, angle constrained, aim assist respecting closest target to cursor. Featuring customizable mechas in terms of color, accessories, limb variants, and armaments. Seamless multiplayer sessions up to four player via port connections, steam, or discord integration. Peer-to-peer client authoritative shooting using raycasts for no host advantage at cost of reduced security. But effects remain accurately telegraphed with synchronized seed, object's relative position, and wall penetration. Other notible impmentations: custom tree data structure with targeting time complexity for duplication, moving, chronologically rebuilding data object nodes to and from save file; object pool for reusing common effects; robust save system allows for restoring of last usable save in event of crash; GPU instancing/ batching parts with same mesh and color; real-time graphics updates using delegates; procedural animation for fluid motion regarding UI elements and player; procedurally generated pve maps and objectives. Development on indefinately on pause as this was an overly ambitious solo project.
