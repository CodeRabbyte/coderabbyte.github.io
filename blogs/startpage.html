Start Page

❮ ❯

Dockerized responsive custom start page web interface solution interacting with libvirt's API to allow for quick and easy access to common virtual machines' functionality along side shortcuts for docker containers. Realtime updates supported with websockets. Constructed with Flask as the backend that renders out a static page, from a json file, with interactive elements controlled via Javascript.

Tools and Technologies:
Libvirt
Python 3
Flask
Javascript
HTML
CSS
Docker
SSH
